# 🏥 AI-Powered Elderly Monitoring System

🚀 Built as a real-time AI healthcare monitoring prototype with a focus on usability, visualization, and intelligent insights.

---

## 📌 Overview

This project is an interactive **real-time dashboard** for monitoring elderly patients using **Human Activity Recognition (HAR)** data.

---

## ⚡ Features

* 📊 **Real-time dashboard** with auto-refresh
* 📈 **Behavior trend analysis** over time
* 🚨 **Alert system** for high-risk events
* 🧑‍⚕️ **Doctor Mode & Admin Mode**
* 📡 **Interactive visualizations** using Plotly
* 🔄 **Live data simulation** for real-time feel

---

## 🛠 Tech Stack

* **Frontend / Dashboard:** Streamlit
* **Data Processing:** Pandas, NumPy
* **Visualization:** Plotly
* **Logic:** Python

---

## 🧠 How It Works

1. Activity data is processed and assigned **behavior scores**
2. Risk levels are classified into:

   * 🟢 Stable
   * 🟡 Moderate
   * 🔴 High Risk
3. Dashboard updates in real-time with:

   * Patient activity trends
   * Anomaly detection
   * Risk alerts

---

## 🎯 Use Case

Designed for healthcare monitoring systems, this dashboard helps:

* Track patient activity patterns
* Detect abnormal behavior early
* Assist doctors and caregivers in decision-making
